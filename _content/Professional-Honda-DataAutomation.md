## Production Reporting System

Excel VBA dashboard integrating PLC data for real-time line performance tracking.

## System Architecture

Modbus TCP connection to 8 PLCs. [https://placehold.co/450/1e40af/FFF?text=Network+Diagram, https://placehold.co/450/1d4ed8/FFF?text=Data+Flow] 5-minute polling interval with SQLite local storage.

## Dashboard Features

OEE calculation, downtime tracking, and Pareto charts. [https://placehold.co/300/2563eb/FFF?text=Main+Dashboard, https://placehold.co/300/3b82f6/FFF?text=Downtime+Analysis, https://placehold.co/300/60a5fa/FFF?text=Trend+Report] Reduced manual data entry from 120 to 15 minutes/day.
